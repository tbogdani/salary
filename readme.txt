Инструкция по использованию программы

1. Заполнить данные по сотрудникам в файле data.xlsx (кол-во отработанных смен, план, фактическое выполнение).
2. В файле st.xlsx указать количество рабочих смен в месяце, проставить грейды сотрудников и кол-во отработанных смен.
3. Запустить файл salary.exe или salary.py, после чего процент выполнения плана, оклад сотрудника, премия и общая сумма зарплаты будут рассчитаны и занесены в файл st.xlsx
